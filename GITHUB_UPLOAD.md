# 📤 Как загрузить DomTextil v2.0 на GitHub (БЕЗ командной строки!)

## 🎯 Способ 1: Через GitHub Web Interface (САМЫЙ БЫСТРЫЙ)

### Шаг 1: Создайте пустой репо на GitHub

1. Откройте https://github.com/new
2. Авторизируйтесь своим GitHub аккаунтом (создайте если нет)
3. Заполните:
   - **Repository name:** `domtextilev2`
   - **Description:** `E-commerce platform v2.0 with PWA`
   - **Visibility:** Public (чтобы Vercel смог подключиться)
4. **НЕ инициализируйте** с README, .gitignore, license
5. Нажмите **Create repository**

### Шаг 2: Загрузите файлы

1. На странице репо нажмите **Add file → Upload files**
2. **Откройте папку** `c:\Users\max\Documents\domtextilev2`
3. **Выделите ВСЕ файлы и папки** (Ctrl+A)
4. **Перетащите их в окно GitHub** или нажмите **choose your files**
5. Внизу напишите сообщение: `DomTextil v2.0 - Initial commit`
6. Нажмите **Commit changes**

✅ **ГОТОВО!** Все файлы загружены на GitHub!

---

## 🎯 Способ 2: GitHub Desktop App (ГРАФИЧЕСКИЙ ИНТЕРФЕЙС)

### Шаг 1: Установите GitHub Desktop

1. Откройте https://desktop.github.com/
2. Скачайте и установите
3. Запустите и авторизируйтесь

### Шаг 2: Добавьте локальный репо

1. В GitHub Desktop: **File → Add Local Repository**
2. Выберите папку: `c:\Users\max\Documents\domtextilev2`
3. Нажмите **Add Repository**

### Шаг 3: Создайте коммит

1. Слева вы увидите все файлы
2. Внизу в поле **Summary** напишите: `DomTextil v2.0 - Initial commit`
3. Нажмите **Commit to main**

### Шаг 4: Загрузите на GitHub

1. Вверху нажмите **Publish repository**
2. Заполните:
   - **Name:** `domtextilev2`
   - **Description:** `E-commerce platform v2.0 with PWA`
   - **Keep this code private:** НЕ ОТМЕЧАЙТЕ (должна быть Public)
3. Нажмите **Publish Repository**

✅ **ГОТОВО!** Ваш код на GitHub!

---

## 🎯 Способ 3: GitHub Web Upload (ШАГ ЗА ШАГОМ С КАРТИНКАМИ)

### Шаг 1: Создайте репо

```
https://github.com/new
  ↓
Repository name: domtextilev2
Description: E-commerce platform v2.0
Visibility: Public
  ↓
Create repository
```

### Шаг 2: Откройте папку на компьютере

```
Откройте: c:\Users\max\Documents\domtextilev2
```

### Шаг 3: Загрузьте файлы

На GitHub репо странице:
```
① Add file ↓
② Upload files ←
③ Выделите ВСЕ файлы (Ctrl+A в папке)
④ Перетащите на GitHub или выберите файлы
⑤ Commit message: "Initial commit"
⑥ Commit changes
```

## 🚀 СЛЕДУЮЩИЙ ШАГ: Развертывание на Vercel

После загрузки на GitHub:

1. Откройте https://vercel.com/new
2. Авторизируйтесь через GitHub
3. Выберите репо `domtextilev2`
4. Нажмите **Import**
5. Добавьте Environment Variables
6. Нажмите **Deploy**

✅ **Ваш сайт в интернете!**
